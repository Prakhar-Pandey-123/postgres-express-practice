<h1>Postgres + Express Practice</h1>

<p>
This repository contains my learning code for using <b>PostgreSQL with Express.js</b>.
It is not a production project. It is made only for practice and understanding.
</p>

<h2>What I practiced</h2>
<ul>
  <li>PostgreSQL basic CRUD operations</li>
  <li>INNER JOIN, LEFT JOIN, RIGHT JOIN, FULL JOIN</li>
  <li>Foreign keys and relations</li>
  <li>Transactions (BEGIN, COMMIT, ROLLBACK)</li>
  <li>Using raw SQL with Express.js</li>
</ul>

<h2>Tech Stack</h2>
<ul>
  <li>Node.js</li>
  <li>Express.js</li>
  <li>PostgreSQL</li>
</ul>

<h2>Project Type</h2>
<p>
Backend only. No frontend is used.
</p>

<h2>How to run</h2>
<ol>
  <li>Clone the repository</li>
  <li>Run <code>npm install</code></li>
  <li>Set up PostgreSQL and database credentials</li>
  <li>Run <code>node index.js</code> or <code>npm start</code></li>
</ol>

<h2>Note</h2>
<p>
This repository is for learning purposes only.
</p>
